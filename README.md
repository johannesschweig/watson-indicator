# watson-indicator
A application indicator for the time-tracking CLI Watson (https://github.com/TailorDev/Watson) for Ubuntu.

There is an icon along with the active project (if any) displayed in the taskbar.  You can stop the current project or restart the last one if there is no active project right now.
